```
1. 紧张，导致常出现拼写错误

2. 对stdin不熟悉，在写测试和实现的时候容易出错

3. 在写最后一个Game类时，测试和实现中的字符串不一样，导致测试出错，在找问题原因时花费较多时间

4. 打字速度慢
```

```

  易出错，打字速度太慢，没有指法，拼写容易出错，对于spyOn的一些用法还不太熟悉，在测试出错的时候找不到错误的原因，以及有些
  粗心，在测试和实现中的字符串输出时格式易出错导致两边匹配不上，测试通过不了，我觉得还是因为自己对这些知识掌握的不好，所以
  必须把易出错的部分再认真学习，另外，要加强打字练习提高打字速度
```
